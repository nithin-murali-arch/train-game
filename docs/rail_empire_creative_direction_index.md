# Rail Empire — Creative Direction Pack

This pack contains three companion documents for the Kimi execution plan:

1. `design_bible.md` — creative canon, tone, scope, and design pillars.
2. `design.md` — implementation-facing game design document.
3. `art_style_guide.md` — visual style, AI/procedural asset pipeline, placeholder rules, and asset production guidance.

Recommended Kimi read order:

1. `design_bible.md`
2. `design.md`
3. `art_style_guide.md`
4. `rail_empire_execution_pack.md`

Kimi should execute one sprint at a time and treat these docs as guardrails, not as permission to add future-scope content early.
